## §A. Независимая перепроверка арифметики (python3, 2026-06-12, с нуля)

**Инф.-теор. граница** (n=10^5, m=10^6, math.lgamma для log2 биномиала):
- n^2 = 10^10, n^2/m = 10^4 ✔
- log2(10^4) = 13.28771237954945 → **13.29** ✔
- аппроксимация m·log2(n^2/m) = 13287712.38 = **13.29e6** ✔
- точное log2 C(10^10, 10^6) = 14730323.99 = 14.730e6 → **14.73** бит/ребро ✔
- аппроксимация (13.29) < точное (14.73) ✔
- разность = 1442611.6 = **1.4426e6** ≈ m·log2 e = 1442695.0 (log2 e=1.4427), отн. ошибка 5.8e-5 ✔ — подтверждает «занижает на положительный член порядка m·log2 e».
- наивный baseline: log2(10^6) = 19.93157 → **19.9** ✔; log2(10^9) = 29.897 → **~30** ✔
- число Каталана: log2 C_n / 2n → 1 (n=10: 0.7018; n=100: 0.9460; n=1000: 0.9921; n=10^4: 0.9990; n=10^5: 0.99987) — **log2 C_n ≈ 2n** ✔
- midpoint 3.3–5.3 = (3.3+5.3)/2 = **4.3** ✔

**k²-дерево, worked example** (матрица 4×4, рёбра {0→1, 2→3, 3→2}, k=2) — реконструировано независимо python3:
```
матрица:    r0[0 1 0 0]  r1[0 0 0 0]  r2[0 0 0 1]  r3[0 0 1 0]
квадрант (0,0)=r0c0,r0c1,r1c0,r1c1 = 0,1,0,0 → есть 1 → бит 1
квадрант (0,1)=r0c2,r0c3,r1c2,r1c3 = 0,0,0,0 → пусто → бит 0 (обрезан)
квадрант (1,0)=r2c0,r2c1,r3c0,r3c1 = 0,0,0,0 → пусто → бит 0 (обрезан)
квадрант (1,1)=r2c2,r2c3,r3c2,r3c3 = 0,1,1,0 → есть 1 → бит 1
T = 1001 ✔
L (только ненулевые): (0,0)=0100, (1,1)=0110 → L = 0100 0110 ✔
```
Полное побитное совпадение с текстом страницы (строки 165–168, 171, 176–177, 180–181). Ошибок нет.
