## §A. Независимая перепроверка арифметики

**Инф.-теор. граница** (Python, n=10^5, m=10^6):
- n^2 = 10^10, n^2/m = 10^4 ✔
- log2(10^4) = 13.287712… → **13.29** ✔
- точное log2 C(10^10, 10^6) = 14730323.99 бит / 10^6 = 14.730… → **14.73** бит/ребро ✔
- аппроксимация (13.29) < точное (14.73); разность = 1.4426… = log2 e ✔ — подтверждает формулировку «занижает на положительный член порядка m·log2 e, это нижняя оценка ведущего порядка, а не равенство».
- наивный baseline: log2(10^6) = 19.9315… → **19.9** ✔

**k²-дерево, worked example** (Python, матрица 4×4, рёбра {0→1, 2→3, 3→2}, k=2):
```
матрица:    r0[0 1 0 0]  r1[0 0 0 0]  r2[0 0 0 1]  r3[0 0 1 0]
квадрант (0,0)=r0c0,r0c1,r1c0,r1c1 = 0,1,0,0 → есть 1 → бит 1
квадрант (0,1)=r0c2,r0c3,r1c2,r1c3 = 0,0,0,0 → пусто → бит 0 (обрезан)
квадрант (1,0)=r2c0,r2c1,r3c0,r3c1 = 0,0,0,0 → пусто → бит 0 (обрезан)
квадрант (1,1)=r2c2,r2c3,r3c2,r3c3 = 0,1,1,0 → есть 1 → бит 1
T = 1001 ✔
L (только ненулевые): (0,0)=0100, (1,1)=0110 → L = 0100 0110 ✔
```
Полное совпадение с текстом страницы (строки 171, 180). Это РИСКОВАННАЯ арифметика — проверена побитно, ошибок нет.
